# SH4 Compatibility tool

**SH4 Compatibility tool** takes binaries (aka addins) for old SH3 Casio Calculators and make them compatible with newer sh4 calculators (power graphic 2).

This tool has originally been created by Ziqumu (aka Alabate) during summer of 2013 for [Plan√®te Casio](https://www.planet-casio.com/Fr/forums/topic12292-1-SH4-compatibility-tool.html).
